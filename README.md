# 홀씨 (Wholesee) - 고립·은둔청년과 함께하는 가족을 위한 AI 서비스

## 프로젝트 개요

홀씨은 히키코모리 자녀를 둔 부모님들을 위한 AI 기반 모바일 상담 및 지원 서비스입니다.
"새로운 대화"의 의미를 담아, 부모님들이 겪는 어려움을 이해하고 실질적인 도움을 제공합니다.

## 타겟 사용자

- 히키코모리(은둔형 외톨이) 자녀를 둔 부모님들
- 자녀와의 소통에 어려움을 겪는 보호자
- 심리적 지원과 실질적 조언이 필요한 가족 구성원

## 기술 스택

- **Frontend Framework**: React 18+
- **Build Tool**: Vite
- **Mobile Framework**: Capacitor.js (iOS/Android 지원)
- **Styling**: Tailwind CSS
- **Language**: TypeScript
- **State Management**: Zustand
- **AI Integration**: OpenAI (gpt5)

## 핵심 기능

### 1. AI 캐릭터와의 대화 (Core Feature) 🧚
**컨셉**: 따뜻한 요정 캐릭터와의 1:1 대화

**주요 기능**:
- 히키코모리 상황에 특화된 AI 캐릭터
- 사용자가 처한 상황을 경청하고 분석
- 공감적이고 실질적인 답변 제공
- 대화 기록 자동 저장 및 누적
- **요정 호감도 시스템** (대화 진행도 시각화)

**AI 캐릭터 특징**:
- 요정 또는 숲속 가이드 역할
- 따뜻하고 비판단적인 태도
- 단계별 질문으로 상황 파악
- 맥락을 이해하고 기억하는 대화
- 사용자의 감정을 먼저 인정

**호감도 시스템** 💝:
- **시각화**: 하트, 별, 또는 게이지바로 표시
- **상승 조건**:
  - 대화를 나눌 때마다 호감도 상승
  - 깊이 있는 대화일수록 더 많이 상승
  - 연속으로 방문하면 보너스
  - 솔직하고 진심 어린 대화 시 추가 상승
- **레벨 단계** (예시):
  - Lv 1-3: 낯선 사이 (첫 만남)
  - Lv 4-6: 알아가는 중 (마음을 열기 시작)
  - Lv 7-9: 친구가 됨 (편하게 대화)
  - Lv 10: 신뢰하는 사이 (커뮤니티 언락!)
- **단계별 변화**:
  - 요정의 말투나 표정 변화
  - 더 깊은 질문과 조언 제공
  - 특별한 메시지/응원 (레벨업 시)
- **UI 위치**: 상단에 요정 아이콘 + 호감도 게이지 항상 표시

### 2. 커뮤니티 언락 시스템 (Progressive Feature) 🏘️
**컨셉**: "요정이 새로운 마을로 초대하는" 판타지 UX

**언락 메커니즘**:
- **호감도 레벨 10 달성** = 커뮤니티 자동 언락
- 언락 조건 상세:
  - 호감도 Lv 10 도달 (필수)
  - 최소 대화 횟수 (예: 10회 이상)
  - 대화 깊이 분석 통과 (단순 인사 X)
  - 최소 사용 기간 (예: 1주일)
  - 감정 정리 단계 달성

**언락 UX 플로우**:
1. 호감도 Lv 10 달성
2. 특별한 애니메이션 (반짝임, 효과음)
3. 요정이 "당신은 이제 준비가 되셨어요. 다른 분들도 만나볼까요?" 메시지
4. "새로운 마을" 초대장 제공 (아름다운 일러스트)
5. 사용자가 선택 (참여 or 나중에)
6. 참여 선택 시 마을 입구 애니메이션

**커뮤니티 기능**:
- 익명 부모님/지인 커뮤니티 공간
- 경험 공유 및 상호 지원
- AI와 충분히 대화한 사람들만 참여 (준비된 상태)
- 안전하고 따뜻한 공간 유지
- 요정이 커뮤니티도 관리하는 설정

## AI 통합 고려사항

### AI 캐릭터 페르소나 🧚
**컨셉**: 오랜 시간 사람들을 돕는 숲속 요정

**톤앤매너**:
- 따뜻하고 공감적인 톤
- 판단하지 않는 중립적 태도
- "함께 걸어가는 친구" 느낌
- 존댓말 사용하되 친근하게
- 희망적이지만 현실적

**캐릭터 설정**:
- 요정 또는 숲속 가이드
- 많은 부모님들의 이야기를 들어온 경험
- 마을(커뮤니티)을 관리하는 역할
- 사용자가 준비되면 마을로 초대

### 프롬프트 설계

**대화 전략**:
- 경청 우선, 섣부른 조언 지양
- 열린 질문으로 상황 파악
- 사용자의 감정을 먼저 인정하고 공감
- 작은 단계부터 구체적인 해결책 제시
- 대화마다 진전을 인정하고 격려

**지식 기반**:
- 히키코모리/은둔형 외톨이 전문 지식
- 한국 문화와 가족 관계 이해
- 부모-자녀 관계 심리학
- 위기 상황 감지 및 대응
- 트라우마 인지 대화법

**대화 분석 (호감도/언락 시스템용)**:
- 대화 깊이 측정 알고리즘 → 호감도 상승폭 결정
- 감정 상태 변화 트래킹
- 주요 이슈와 패턴 파악
- 진전도 체크 (커뮤니티 준비도)
- 호감도 레벨에 따른 AI 반응 조정

### 안전장치
- 자해/자살 위험 감지 시 즉시 전문 기관 연결
- 의료적 진단/치료는 전문가 권유
- 법적 문제는 전문가 상담 안내
- 커뮤니티 언락 전 준비도 평가
- 위기 상황에서는 언락 보류
